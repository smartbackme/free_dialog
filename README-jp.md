## free_dialog

free_dialog は極めて簡単なdialogスキームで、簡単に構成を変更し、使用効菓を達成することができます。

### 開始

```dart
dependencies:
  free_dialog: ^0.0.1
```

### 使用方法：

1. 多言語環境構成（デフォルトは英語）

```dart
      localizationsDelegates: [
        FreeLocalizations.delegate],
      supportedLocales: const [
        Locale('en'),
        Locale('zh'),
        Locale('jp'),
      ],

```

2. 簡単に使う

demoを参照してください。
